# MyGame

My journey toward learning game development using Python.

## `pdm` Commands

Prerequisites:
- [pdm](https://pdm-project.org/en/latest/)

```sh
# clone repo
git clone git@github.com:ramonmeza/my_game.git

# install environment
pdm install

# commands
pdm run build-docs
pdm run format
pdm run game
pdm run lint
```
